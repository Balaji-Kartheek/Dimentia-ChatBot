# 📖 Usage Guide - Dementia Chatbot

This guide explains how to use the Dementia Chatbot effectively for both people with dementia and their caregivers.

## 🔑 Getting Started

### Login
1. Open the application in your web browser
2. Choose your role:
   - **Person with Dementia**: Username `user`, Password `dementia123`
   - **Caregiver**: Username `caregiver`, Password `caregiver123`

### First Time Setup
1. **Select Language**: Choose your preferred language from the sidebar
2. **Test Audio**: Go to Settings and test microphone and speakers
3. **Add First Memory**: Start by adding a simple memory

## 👤 For People with Dementia

### 🧩 Adding Memories

#### Voice Method (Recommended)
1. Go to **"Add Memory"** page
2. Select **"🎤 Voice Recording"**
3. Click the microphone button
4. Speak clearly: *"I take my blood pressure medication every morning at 8 AM"*
5. Click **"Process Recording"**
6. Review the transcribed text
7. Click **"Save Memory"**

#### Text Method
1. Go to **"Add Memory"** page
2. Select **"✏️ Text Input"**
3. Type your memory: *"I have a doctor's appointment on March 15th at 2 PM"*
4. Review extracted information (dates, times, etc.)
5. Click **"Save Memory"**

#### Memory Examples
- **Medications**: *"I take my diabetes medicine twice a day"*
- **Appointments**: *"My dentist visit is next Tuesday at 3 PM"*
- **Family**: *"My daughter Sarah lives in New York"*
- **Routines**: *"I walk my dog every morning at 7 AM"*
- **Locations**: *"My favorite restaurant is Luigi's on Main Street"*

### 🔍 Asking Questions

#### Voice Questions
1. Go to **"Ask Assistant"** page
2. Select **"🎤 Voice Question"**
3. Record your question: *"When do I take my medication?"*
4. Click **"Process Question"**
5. Listen to the spoken response
6. Review sources used

#### Text Questions
1. Go to **"Ask Assistant"** page
2. Select **"✏️ Text Question"**
3. Type your question
4. Use quick question buttons for common queries
5. Click **"Get Answer"**

#### Question Examples
- *"When is my next appointment?"*
- *"What medications do I take?"*
- *"Tell me about my family"*
- *"Where do I live?"*
- *"What did I do yesterday?"*

### 🌐 Language Features

#### Changing Language
1. Use the language selector in the sidebar
2. Available languages: English, Hindi, Tamil, Spanish, French, German
3. Interface and voice recognition will adapt

#### Multilingual Usage
- Ask questions in your preferred language
- Add memories in any supported language
- Get responses in the same language

### ✅ Memory Review

#### Review Process
1. Go to **"Memory Review"** tab
2. Filter memories by:
   - Language
   - Trust level (unverified/trusted/flagged)
   - Source (voice/text/imported)
3. Review each memory for accuracy
4. Take actions:
   - **✓ Verify**: Mark as trusted
   - **⚠️ Flag**: Mark for attention
   - **🗑️ Delete**: Remove incorrect memory

#### Verification Guidelines
- **Verify** accurate medical information
- **Flag** unclear or potentially incorrect memories
- **Delete** completely wrong information
- Consider context and user's condition

## 💡 Best Practices

### For People with Dementia

#### Adding Memories
- **Be specific**: Include dates, times, and details
- **Use simple language**: Avoid complex sentences
- **Add regularly**: Update information as it changes
- **Include context**: Help the AI understand better

#### Asking Questions
- **Be patient**: Wait for the system to process
- **Ask clearly**: Use simple, direct questions
- **Try different phrasings**: If first attempt doesn't work
- **Check sources**: Review where information came from

### For Caregivers

#### Memory Management
- **Review regularly**: Check new memories frequently
- **Verify medical info**: Ensure accuracy of health data
- **Update as needed**: Keep information current
- **Monitor patterns**: Watch for changes in user behavior

#### User Support
- **Encourage use**: Help establish routine
- **Explain features**: Guide through new functionality
- **Provide feedback**: Report issues or suggestions
- **Maintain privacy**: Respect user's personal information

## 🚨 Important Notes

### Privacy and Security
- **All data stays local**: Nothing is sent to external servers
- **Encrypted storage**: Memories are encrypted on your device
- **Access control**: Separate interfaces for users and caregivers
- **Activity logging**: Track all interactions for safety

### Limitations
- **Internet required**: Only for initial setup and updates
- **Local processing**: Performance depends on your device
- **Language support**: Limited to configured languages
- **Model dependency**: Requires Ollama and Whisper models

### Emergency Situations
- **Medical emergencies**: Don't rely solely on the chatbot
- **Critical information**: Always verify with healthcare providers
- **Data backup**: Keep regular backups of important information
- **System failures**: Have alternative memory aids available

## 🔧 Troubleshooting

### Common User Issues

#### Voice Not Working
1. Check microphone permissions
2. Test in Settings page
3. Ensure clear speech
4. Try text input instead

#### Questions Not Answered
1. Check if relevant memories exist
2. Try different question phrasing
3. Add more specific memories
4. Verify system is running

#### Language Issues
1. Confirm language selection
2. Check if language is supported
3. Restart application
4. Verify model availability

### Caregiver Issues

#### Cannot Access Console
1. Verify caregiver login credentials
2. Check user role assignment
3. Restart application
4. Clear browser cache

#### Memory Review Problems
1. Check database permissions
2. Verify system components
3. Rebuild memory index
4. Check system logs
